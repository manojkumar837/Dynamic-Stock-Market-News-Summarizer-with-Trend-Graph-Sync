# 📈 Dynamic Stock Market News Summarizer with Trend Graph Sync

An AI-powered stock market analytics dashboard that combines real-time stock prices, financial news, sentiment analysis, and interactive trend visualization into a single platform. The application enables investors and analysts to monitor market movements, understand news sentiment, and make informed decisions using synchronized stock charts and AI-generated news summaries.

---

## 🚀 Features

- 📊 Real-time stock price visualization
- 📈 Interactive trend graphs using Plotly
- 📰 Live financial news aggregation
- 🤖 AI-powered news summarization
- 😊 Sentiment analysis (Positive, Neutral, Negative)
- 📅 Custom date range selection
- 📉 Technical indicators (Moving Average, RSI, MACD, Bollinger Bands)
- 🔍 Search stocks by ticker symbol
- ⚡ Fast and interactive Streamlit dashboard
- 📱 Responsive and user-friendly interface

## ⚙️ Workflow

1. Fetch historical and real-time stock data using Yahoo Finance.
2. Collect the latest financial news from online sources.
3. Perform sentiment analysis on news headlines.
4. Generate AI-powered news summaries.
5. Synchronize stock trend graphs with market news.
6. Display KPIs, technical indicators, and interactive visualizations.
